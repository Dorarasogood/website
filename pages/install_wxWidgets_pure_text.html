//////////////////////[==wxWidgets安裝步驟==]///////////////////////


(步驟1)

下載wxWidgets，去https://www.wxwidgets.org/downloads/，找到Latest Stable Release區塊，在Source Code欄位那邊，點選Windows ZIP，(此版本是wxWidgets-3.0.4.zip)

(步驟2)

將下載下來的檔案(wxWidgets-3.0.4.zip)解壓縮到C槽(不含有空白符號的資料夾目錄，Do not install into a path that contains spaces.)

(步驟3)

用ubuntu bash(也可在vs code開bash)打開wxWidgets-3.0.4檔案

(步驟4)

下指令(在wxWidgets-3.0.4檔案最外層，cd /path/to/wxWidgets-3.0.x # adapt path as needed)

$sudo apt-get update

$sudo apt-get install libwxgtk3.0-dev

$ sudo apt-get install libgtk-3-dev build-essential checkinstall

$ mkdir gtk-build
$ cd gtk-build/

$ ../configure --enable-unicode --enable-debug

$make

$sudo make install

(步驟5)

確認是否正確安裝

$wx-config --version

(步驟6)

安裝server(請看下面的VcXsrv教學)

(步驟7)

server(VcXsrv)開啟後，下指令(原來地方)

$echo "export DISPLAY=localhost:0.0" >> ~/.bashrc

$. ~/.bashrc

(步驟8)

測試server是否正確開啟

$sudo apt-get install x11-apps

$export DISPLAY=:0

$xeyes

(步驟9)

測試內建小程式

(去gtk-build/samples/minimal)

$cd samples/minimal
$make
$./minimal

(步驟10)

撰寫自己的程式

寫好程式後，在你的程式檔案目錄下，下指令(XXX:代表檔案名稱)

$g++ XXX.cpp `wx-config --cxxflags --libs` -o XXX

$export DISPLAY=:0

./XXX



完成!!!
